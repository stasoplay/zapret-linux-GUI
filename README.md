# Как пользоваться
Первым делом нужно запустить скрипт main_script.sh как и в дефолте, а ещё проверить, совпадает ли путь к папке с файлами. Изначально он лежит в /home/App. Если у вас путь отличается, зайдите в zapret.sh и Window.sh и в начале измените путь

Запустив Window.sh, вы запустите само окно.
