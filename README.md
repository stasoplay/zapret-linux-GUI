# Форк zapret с недоGUI и Warp'ом
Сделал в основном для себя но решил выкласть это как форк

# Как пользоваться
Первым делом нужно проверить, совпадает ли путь к папке с файлами. Изначально он лежит в /home/App. Если у вас путь отличается, зайдите в zapret.sh и Window.sh и в начале измените путь
